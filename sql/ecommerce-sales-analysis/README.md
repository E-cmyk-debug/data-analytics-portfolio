# E-Commerce Sales Analysis (SQL Project)

## Overview

This project analyses an e-commerce dataset to extract meaningful business insights related to sales performance, customer behaviour, and product trends.

The analysis is performed using SQL in MySQL.

## Objectives

* Calculate total sales and profit
* Identify top-performing regions and products
* Analyse monthly and yearly sales trends
* Determine category-wise contribution
* Identify loss-making orders

## Tools Used

* MySQL Workbench

## Project Structure

* `dataset.csv`
* `queries.sql`
* `screenshots/`

## 📈 Key Insights

* Top regions contribute most revenue
* Sales follow monthly patterns
* Few products drive majority sales
* Some orders generate losses
