## 1. Project Title

**Profit Optimization & Cost Analysis (Excel)**

## 2. Overview

This project analyzes a global e-commerce dataset using Microsoft Excel to understand key factors affecting business profitability.
The goal is to identify profit leakages and provide actionable insights to improve overall profit margins.

## 3. Dataset Description

* Source: Kaggle (Global E-Commerce Dataset)
* The dataset contains transaction-level data, including:

  * Order_ID
  * Order_Date
  * Customer_Name
  * Customer_Segment
  * Country
  * Region
  * Product_Category
  * Product_Name
  * Quantity
  * Unit_Price
  * Discount_Percent
  * Total_Sales
  * Shipping_Cost
  * Profit
  * Payment_Method

## 4. Business Questions / Problem Statement

* Where is the business losing money?
* Which regions and product categories are underperforming?
* How do discounts impact profitability?
* Does shipping cost significantly affect profit margins?
* Which categories contribute most to loss-making orders?

## 5. Tools Used

* Microsoft Excel

  * Power Query (Data Cleaning & Transformation)
  * Pivot Tables
  * Pivot Charts
  * Dashboard Design

## 6. Approach / Process

* Cleaned and transformed data using Power Query
* Created calculated columns such as Profit Margin %, Discount Category, Shipping Cost Ratio, and Loss Flag
* Performed exploratory data analysis using Pivot Tables
* Analysed profit drivers including discount impact, shipping cost, and loss concentration
* Built an interactive dashboard to visualise key insights

## 7. Key Insights

* High discounts significantly reduce profit margins
* Shipping costs heavily impact profitability in South America and Middle East & Africa
* Office Supplies category contributes the majority of loss-making orders
* Profitability varies significantly across regions, with some regions consistently underperforming

## 8. Conclusion

The analysis identifies key drivers of profit leakage, including excessive discounting, high logistics costs, and inefficient product categories.
These insights can help businesses optimise pricing strategies, reduce operational costs, and improve overall profitability.
